Minecraft Console Client (MCC) is a lightweight, cross-platform, open-source TUI client for Minecraft Java Edition. It lets you connect to any Minecraft Java server, send chat and commands, and receive messages — all without opening the main game. MCC also ships with 28 built-in automation bots and a full C# scripting API for building your own.

## Get up and running

<Steps>
  <Step title="Install MCC">
    Run the one-liner installer for your platform:

    <CodeGroup>
    ```bash Linux / macOS
    curl -fsSL https://mccteam.github.io/install.sh | sh
    ```
    ```powershell Windows (PowerShell)
    iwr -useb https://mccteam.github.io/install.ps1 | iex
    ```
    </CodeGroup>
  </Step>
  <Step title="Run the client">
    Launch MCC from your terminal. On first run it creates a `MinecraftClient.ini` config file:

    <CodeGroup>
    ```bash Linux / macOS
    ./MinecraftClient
    ```
    ```powershell Windows
    .\MinecraftClient.exe
    ```
    </CodeGroup>
  </Step>
  <Step title="Connect to a server">
    You can pass your credentials and server directly on the command line:

    ```bash
    ./MinecraftClient your@email.com yourpassword play.example.com
    ```

    Or configure `MinecraftClient.ini` with your account and server, then just run MCC.
  </Step>
  <Step title="Automate with bots or scripts">
    Enable built-in bots in the config file, or create your own with a simple text script or C# bot file.
  </Step>
</Steps>
